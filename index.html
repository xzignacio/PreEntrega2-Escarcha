Que es un Setup

Un setup es el conjunto de elementos de hardware, periféricos y accesorios que se necesitan para tener una buena experiencia de juego o trabajo. Esto incluye desde accesorios fundamentales como lo son pc, teclado, mouse, mousepad, monitor,etc. Pero también se incluyen elementos que dejan en segundo plano su practicidad a cambio de mayor comodidad para sentirnos cómodos en el setup y poder mejorar nuestro humor o estados de animo a la hora de pasar largas horas en el set up.

A lo largo del tiempo se crearon diversas tendencias para nuestros setups o espacios de trabajo, adoptando estilos mas iluminados y dedicado al gaming, como estilos mas minimalistas para poder aprovechar el espacio y dar una aparencia limpia, como setups dedicados al trabajo y productividad.

Como mejorar un Setup

Como se menciono anteriormente, los set ups cuentan con diversos elementos que hacen que nuestro espacio de trabajo sea mas cómodo y efciente, como elementos decorativos que hacen que nuestro espacio de trabajo sea mas cómodo y amigable para estar largas horas en un mismo lugar, como el uso de plantas que hacen que nuestro espacio tenga color y se sienta mayor confort, como el uso de alfombras para que nuestros pies no pasen frio y nuestro suelo no se desgaste por el movimiento de la silla, como soportes de monitores para poder tener una buena postura y cuidar nuestra vista.

Estos distintos elementos si bien no son de vital importancia hacen que nuestro cerebro asocie estos elementos con relajación y paz.
